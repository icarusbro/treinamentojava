Para preparar o ambiente local para testes seguir os passos:
1-) Baixar o Xampp (Windows) instala-lo e iniciar uma base de dados mySQL.
2-) Utilizar um gerenciador de conexão mySQL e rodar os comandos encontrados no script preparo.sql
3-) Importar o projeto treinamento-backend no intellij e configurar uma maven run com o seguinte goal spring-boot:run
4-) Iniciar a maven run em modo run ou debug
